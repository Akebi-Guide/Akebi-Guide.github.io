1.请不要倒卖，此项目为 Github 的开源项目！倒卖开源项目是可耻的！

2.请不要打深渊！绝对！

3.开锚点、开宝箱、拿神瞳请至少间隔30秒，过秘境请至少间隔10秒，总之，任何涉及奖励的事包括拾取物品都要间隔时间！

4.请尽量不要去陌生人世界破坏陌生人的游戏体验！原神对在线世界是没有检测的，但被举报得不偿失！

5.活动分数不宜太多！绝对不要开Kill Aura（范围死亡）过活动！
